# Particles Ultimate

This is implementation of simple rotating particles visual effect

## Installation & Usage

**Clone repository**
```bash
git clone https://github.com/p4sttt/lazarus-particles-ultimate.git
cd ./lazarus-particles-ultimate/bin
```

**Run binary file**
```bash
./particles
```
